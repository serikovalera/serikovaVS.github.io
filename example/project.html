Mobile application "Quote"

Stack: Flet, Dart


Source Code

The app shows the quote of the day. After clicking the "Get" button, a quote appears on the screen and the button becomes inactive. A timer is started that counts down the time until the quote can be received again.
The code is written in Dart using the Flutter framework.
